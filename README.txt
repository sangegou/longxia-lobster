龙虾批发开单小程序 — 使用说明
================================

【打开方式】
  用浏览器打开 index.html 文件即可使用。
  推荐添加到手机主屏幕，像 App 一样使用（见下方说明）。

【添加到手机主屏幕】

  📱 iPhone / iPad (Safari)
    1. 用 Safari 浏览器打开 index.html
    2. 点击底部中间的"分享"按钮（方框带箭头图标）
    3. 向下滑动找到"添加到主屏幕"
    4. 点击右上角"添加"即可

  📱 Android (Chrome / 系统浏览器)
    1. 用浏览器打开 index.html
    2. 点击右上角菜单（⋮ 三个点）
    3. 选择"添加到主屏幕"
    4. 点击"添加"即可

  📱 微信中打开
    1. 微信发送文件给自己或使用"文件传输助手"
    2. 点击文件 → 选择"用其他应用打开" → 选择浏览器
    3. 在浏览器中按上述步骤添加到主屏幕

【基本流程】
  1. 定价：每天开始前，先进入"定价"页面，输入各规格单价，确认价格。
  2. 开单：进入"开单"页面，输入客户姓名和每种规格的重量，自动计算金额。
     每个规格旁会显示当天单价，截图分享时单价也会包含在账单中。
  3. 汇总：查看每天的收购总量和总金额。

【四个页面说明】

  💰 定价
    - 显示当天日期（日期自动更新）
    - 公虾和母虾分类显示
    - 输入每个规格对应的单价（元/斤）
    - 点击"确认今日价格"，弹出确认框核对后锁定
    - 如需修改，点击"修改价格"（会提示当前已有开单数）

  📝 开单
    - 每个规格旁显示当日单价（@XX元/斤），截图时客户也能看到单价
    - 先确认当日价格已设置，否则无法开单
    - 输入客户姓名（支持历史记录自动补全）
    - 输入各规格重量，右边自动显示该规格金额
    - 重量支持加法输入，如：11.3+11.5+8.2
    - 底部显示总重量和总金额
    - 点击"完成开单"后显示账单明细（含单价）
    - 可截图分享给客户，也可修改或删除已开单据

  📊 汇总
    - 查看每天各规格的收购总量和总金额
    - 显示当日全部账单明细
    - 支持"复制汇总"文本，方便微信发送

  ⚙️ 设置
    - 规格管理：添加/编辑/删除/排序规格
    - 数据备份：导出全部数据为 JSON 文件
    - 数据恢复：导入之前备份的 JSON 文件
    - 字体大小：小/中/大三档切换

【数据存储】
  所有数据保存在浏览器本地，不会上传到任何服务器。
  建议定期使用"导出数据"功能备份。
  更换手机或浏览器时，使用"导入数据"恢复。

【注意事项】
  - 价格确认后即锁定，修改需要二次确认（且不影响已开单据）
  - 开单后的单据会保留创建时的价格，即使后续修改价格也不影响历史单据
  - 可通过日期左右箭头查看历史记录
  - 截图账单明细中包含：规格、单价、重量、金额，方便客户核对
